Import Adobe InDesign (IDML) files into CE.SDK — the browser-based Creative Editor SDK from IMG.LY. Convert existing InDesign layouts into fully editable scenes for design, social media, marketing, and document automation workflows, so your users can keep working with the templates your team already uses in InDesign.

CE.SDK is a drop-in creative suite for web and mobile apps: photo editing, video editing, design templates, and headless rendering. The InDesign Importer is one of several format bridges that let you bring real-world design assets into CE.SDK-powered products.
